Hi! This is a kid-friendly series I'm writing for my son **Anish**, who is 11. Every day is a small, self-contained lesson — wire something up, type some code, watch it come alive, and learn one new idea.

No prior coding needed. No prior electronics needed. Just an Arduino Uno kit and some curiosity.

## What you will build

Over 26 days, you go from blinking a tiny light to designing a smart gadget that listens, watches, talks, and moves.

### The journey

**Getting started (Days 1–6)** — lights, loops, and your first real project

- [**Day 1**](/arduino/day-1-intro-to-arduino/) — Meet the Arduino, install the IDE, blink the built-in LED
- [**Day 2**](/arduino/day-2-external-led-breadboard/) — Wire up your own LED on a breadboard
- [**Day 3**](/arduino/day-3-button-input/) — Add a button, make the LED listen to you
- [**Day 4**](/arduino/day-4-variables-and-loops/) — Variables and `for` loops — write less, do more
- [**Day 5**](/arduino/day-5-led-sequences/) — Three LEDs dancing, chasing, and flickering
- [**Day 6**](/arduino/day-6-traffic-light/) — Build a real traffic light with a pedestrian button

**Making Arduino think (Days 7–10)** — inputs, sensors, and decisions

- [**Day 7**](/arduino/day-7-input-pullup/) — The easier way to wire a button (`INPUT_PULLUP`)
- [**Day 8**](/arduino/day-8-potentiometer/) — Potentiometer + Serial Monitor + dimming an LED
- [**Day 9**](/arduino/day-9-light-sensor/) — Light sensor (LDR) — Arduino sees the world
- [**Day 10**](/arduino/day-10-smart-night-light/) — Build a Smart Night Light

**Movement and sound (Days 11–14)** — motors and buzzers

- [**Day 11**](/arduino/day-11-buzzer/) — Buzzer — make noise, then a melody
- [**Day 12**](/arduino/day-12-servo-motor/) — Servo motor — rotate to any angle
- [**Day 13**](/arduino/day-13-dc-motor/) — DC motor with a transistor and speed control
- [**Day 14**](/arduino/day-14-auto-fan/) — Build an Auto Fan that turns on when it is hot

**Talking back (Days 15–18)** — LCD display

- [**Day 15**](/arduino/day-15-lcd-basics/) — Hook up a 16×2 LCD and say "Hello Anish!"
- [**Day 16**](/arduino/day-16-lcd-sensor/) — Show live sensor values on the screen
- [**Day 17**](/arduino/day-17-lcd-menu/) — Build a menu you can scroll through with buttons
- [**Day 18**](/arduino/day-18-weather-station/) — Build a mini Weather Station

**Smart control (Days 19–22)** — wireless and sound

- [**Day 19**](/arduino/day-19-ir-remote/) — Control Arduino with a TV remote (infrared)
- [**Day 20**](/arduino/day-20-sound-sensor/) — Detect claps with a sound sensor
- [**Day 21**](/arduino/day-21-ir-and-sound/) — Combine IR and sound into dual-mode control
- [**Day 22**](/arduino/day-22-smart-room/) — Build a Smart Room System

**Final project (Days 23–26)** — design, build, debug, demo

- [**Day 23**](/arduino/day-23-project-planning/) — Pick a project, sketch the system
- [**Day 24**](/arduino/day-24-build-core/) — Build the core features
- [**Day 25**](/arduino/day-25-integration-debug/) — Integrate, debug, and polish
- [**Day 26**](/arduino/day-26-demo-day/) — Demo Day — show it off to family

## How to follow along

Read one day at a time. Each post has:

- What you need (components)
- A circuit diagram showing exactly which pin connects to what
- The full code, copy-pasteable
- Detailed explanation of every new idea
- Challenges to try on your own

Start with **[Day 1](/arduino/day-1-intro-to-arduino/)**.
